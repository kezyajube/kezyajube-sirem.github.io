SIREM - Sistema Repositório Multiletramento

O Projeto

O Sistema Repositório Multiletramento (SIREM) é um projeto para criar um repositório para alunos do EMI (Ensino Médio Integrado) em informática, nas disciplinas Litetura e Informática.

Recursos Principais

  • Repositório de recursos educacionais
  • Integração de múltiplas linguagens
  • Facilidade de busca e navegação
